------- Stage 1 — MapSPAM 처리 -------

+-------------------------------------------------------------+
|  process_mapspam.py                                          |
+-------------------------------------------------------------+
|  IN  <-- prep/data/mapspam-2020/csv/                         |
|           |- spam2020V2r0_global_A_TA.csv (~265MB)           |
|           |- spam2020V2r0_global_A_TI.csv                    |
|           |- spam2020V2r0_global_A_TR.csv                    |
|           |- spam2020V2r0_global_H_TA.csv                    |
|           |- ... (총 12개)                                    |
|                                                              |
|  DO  -- 모든 글로벌 CSV 읽음                                   |
|         -> FIPS0='US' 필터링                                  |
|         -> 작물 컬럼들 (BANA, MAIZ, SOYB...) join             |
|         -> all-zero 컬럼 제거                                 |
|                                                              |
|  OUT --> prep/data/mapspam-2020/mapspam2020_USA_wide.csv     |
|           (75,835 rows × 361 cols)  ✓ 완료                   |
+-------------------------------------------------------------+


------- Stage 2 — 토양 프로파일 -------

+-------------------------------------------------------------+
|  parse_sol_to_csv.py                                         |
+-------------------------------------------------------------+
|  IN  <-- prep/US.SOL                  <-- <null>             |
|           (DSSAT 토양 파일,              (출처 불명, ISRIC      |
|            12행 × N profile)             SoilGrids v2 + HC27 |
|                                          기반으로 별도 생성)   |
|                                                              |
|       <-- prep/cell_id.py             <-- <null>             |
|           (헬퍼, lat/lon -> CELL5M)      (직접 작성 필요)      |
|                                                              |
|  DO  -- US.SOL 12행 단위로 파싱                                |
|         -> 각 프로파일의 lat/lon 추출                          |
|         -> cell_id로 CELL5M 변환                              |
|                                                              |
|  OUT --> prep/US_SOL_CELL5M.csv                              |
|           (CELL5M, SoilProfileID, SoilProfile)               |
+-------------------------------------------------------------+


------- Stage 3 — 토양 뿌리 깊이 -------

+-------------------------------------------------------------+
|  extract_resdept.py                                          |
+-------------------------------------------------------------+
|  IN  <-- prep/data/mapspam-2020/mapspam2020_USA_wide.csv     |
|           (Stage 1 출력에서 CELL5M, x, y만 사용)              |
|                                                              |
|       <-- USDA SDA REST API                                  |
|           sdmdataaccess.nrcs.usda.gov                        |
|           (공개 API, 키 불필요)                                |
|                                                              |
|  DO  -- 100개씩 배치로 좌표 -> mukey 쿼리                      |
|         -> mukey -> resdept_r 쿼리                            |
|         -> 캐시: prep/sda_cache/s1_*.json, s2_*.json         |
|         -> 약 759 배치, 40~60분 소요                           |
|                                                              |
|  OUT --> prep/rooting_depth_CELL5M.csv                       |
|           (CELL5M, resdept_r)  (진행 중)                     |
+-------------------------------------------------------------+


------- Stage 4 — 식재일 -------

+-------------------------------------------------------------+
|  parse_plant_dates.py                  (Soybean용)           |
+-------------------------------------------------------------+
|  IN  <-- prep/*.crop.calendar.fill/plant.asc  <-- <null>     |
|           (GAEZ ASCII raster, 5분각)              (정확한 버전 |
|                                                    불명, 아마  |
|                                                    GAEZ v3.0) |
|                                                              |
|  DO  -- 각 작물 폴더의 plant.asc 읽음                          |
|         -> 6줄 헤더 건너뛰기                                   |
|         -> cell_id 별 DOY 값 모음                             |
|                                                              |
|  OUT --> prep/plant_dates_CELL5M.csv                         |
|           (CELL5M, X, Y, Maize, Soybeans, ...)               |
+-------------------------------------------------------------+

+-------------------------------------------------------------+
|  [!] nass_maize_planting.py            (Maize용)             |
|  [!] smooth_maize_planting.py          (Gaussian smoothing)   |
+-------------------------------------------------------------+
|  X 파일 자체가 prep/ 폴더에 없음                               |
|  IN  <-- <null> NASS API + USDA Ag Handbook 628              |
|  DO  -- <null>                                               |
|  OUT --> <null>                                              |
+-------------------------------------------------------------+


------- Stage 4.5 — Wide -> Long 변환 (누락) -------

+-------------------------------------------------------------+
|  [!] wide-to-long pivot 스크립트 누락                         |
+-------------------------------------------------------------+
|  IN  <-- mapspam2020_USA_wide.csv                            |
|       <-- US_SOL_CELL5M.csv                                  |
|       <-- rooting_depth_CELL5M.csv                           |
|       <-- plant_dates_CELL5M.csv                             |
|                                                              |
|  DO  -- ≥25 ha MZ/SB 필터링 (75,835 -> 47,707 rows)          |
|         -> wide -> long 변환                                  |
|         -> MZ/SB 두 작물 join (MZ,SB 콤마 페어 생성)          |
|                                                              |
|  OUT --> maiz_soyb_25ha_long_soil_depth_plant_nass.csv       |
|           ^ 5단계 입력 파일명에 해당하는 파일                  |
|  X 이 변환 단계의 .py 파일이 prep/에 없음                     |
+-------------------------------------------------------------+


------- Stage 5 — 실제 N 시비량 -------

+-------------------------------------------------------------+
|  nass_nfert.py                                               |
+-------------------------------------------------------------+
|  IN  <-- prep/maiz_soyb_25ha_long_soil_depth_plant_nass.csv  |
|           (Stage 4.5 출력 — 누락 [!])                         |
|       <-- prep/mapspam2020_USA_wide.csv                      |
|           (CELL5M -> state 매핑용)                            |
|                                                              |
|       <-- USDA NASS QuickStats API                           |
|           quickstats.nass.usda.gov                           |
|           (FREE 키 필요, 없으면 하드코드 fallback 사용)        |
|       <-- FALLBACK_N_MZ, FALLBACK_N_SB                       |
|           (스크립트 내장 41주 데이터)                          |
|                                                              |
|  DO  -- lb/acre × 1.12085 = kg/ha 변환                       |
|         -> 각 행에 state 기반 N rate 할당                     |
|         -> MZ,SB 페어는 콤마로 (155.0,5.6 형태)               |
|                                                              |
|  OUT --> prep/maiz_soyb_25ha_long_soil_depth_plant_nass_nfert.csv |
+-------------------------------------------------------------+


------- Stage 6 — 관개 분류 -------

+-------------------------------------------------------------+
|  mapspam_irrigation.py                                       |
+-------------------------------------------------------------+
|  IN  <-- Stage 5 출력 (...nass_nfert.csv)                    |
|       <-- prep/mapspam2020_USA_wide.csv                      |
|           (MAIZ_A_TI, MAIZ_A_TR, SOYB_A_TI, SOYB_A_TR 사용)  |
|                                                              |
|  DO  -- TI/(TI+TR) > 0.5 -> "I"                              |
|         else -> "R"                                          |
|                                                              |
|  OUT --> ...nass_nfert_irr.csv                               |
+-------------------------------------------------------------+


------- Stage 7 — 식재 밀도 -------

+-------------------------------------------------------------+
|  nass_planting_density.py                                    |
+-------------------------------------------------------------+
|  IN  <-- Stage 6 출력 (...nass_nfert_irr.csv)                |
|       <-- prep/mapspam2020_USA_wide.csv                      |
|       <-- USDA NASS Objective Yield Survey API               |
|           (10개 주 데이터, 옵션)                               |
|       <-- FALLBACK_DENS_MZ {state: {I, R}}                   |
|       <-- FALLBACK_DENS_SB {state: density}                  |
|       <-- IRR_OVERRIDES_MZ {Kansas: 7.5}                     |
|           (모두 스크립트 내장)                                 |
|                                                              |
|  DO  -- plants/acre ÷ 4046.86 = plants/m²                    |
|         -> WaterSupply 컬럼 보고 I/R별 값 선택 (MZ만)         |
|         -> SB는 관개 무관 같은 값                             |
|                                                              |
|  OUT --> ...nass_nfert_irr_dens.csv                          |
+-------------------------------------------------------------+


------- Stage 8 — 추천 N 시비량 -------

+-------------------------------------------------------------+
|  nfert_recommended.py                                        |
+-------------------------------------------------------------+
|  IN  <-- Stage 7 출력 (...nass_nfert_irr_dens.csv)           |
|       <-- REC_N_MZ_LB                                        |
|           {state: {I, R}}  (스크립트 내장)                    |
|           - 7 MRTN주: Sawyer et al. 2006 / state bulletins   |
|           - 나머지: UNL, KSU, TAMU, NCSU, UGA, UC Davis      |
|       <-- 콩 추천량 = 0 kg/ha (BNF로 자체 공급)               |
|                                                              |
|  DO  -- 기존 NFertRate -> NFertRateAct 이름 변경             |
|         -> NFertRateRec 컬럼 추가                            |
|         -> WaterSupply 보고 I/R별 추천량 적용                 |
|                                                              |
|  OUT --> ...nass_nfert_irr_dens_nrec.csv                     |
|           = 최종 unit-information CSV (47,707 rows × 14 cols)|
+-------------------------------------------------------------+
